# fast_commandsys_cpp
## A simple fast_commandsys rewritten in C++ (for more cross platform support)


### How to use 

#### Create a command:
```
auto sys = new CommandSystem::Command;
auto kek = sys->add("lel", kekw);
kek.run();
```
